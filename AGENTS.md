Do not reintroduce Supabase. This project pivoted to Convex and all Supabase traces were removed.

## Commands

Run from the project root:

```bash
pnpm install
pnpm dev
pnpm test
pnpm lint
pnpm build
```

Convex setup:

```bash
pnpm exec convex dev
```

Manual ingest:

```bash
curl -H "authorization: Bearer $CRON_SECRET" http://localhost:3002/api/ingest/run
```

## Design Rules

The UI was redesigned using the `design-taste-frontend` skill.

Preserve:

- Light neutral trading-desk aesthetic
- One teal accent
- Phosphor icons
- Monospace numbers
- No emojis
- Mobile-first single-column fallback
- `min-h-[100dvh]` for full-height screens

Avoid:

- Generic dark cards
- Purple/blue AI gradients
- Lucide icons
- Supabase references
- Overly centered hero layouts
- Fake-looking rounded card grids

## Source Behavior

Polymarket is public/no-key.

Kalshi may require free credentials.

News/context sources are optional and quota-aware:

- RSS works with `RSS_FEEDS`
- GNews requires `GNEWS_API_KEY`
- NewsAPI requires `NEWSAPI_KEY` and is disabled in production unless explicitly enabled
- Reddit currently uses search links, not a full OAuth read integration

The app shows sample data until Convex is configured and ingestion runs.
